Update History

  

V2.5.0 / 14-October-2013

 

Main Changes

•  Suppression of STM8L devices with charge-transfer (CT) acquisition.

   Note: These devices and the CT acquisition are now supported in the new STM8L STMTouch Library.

•  Only STM8S/A devices with RC acquisition are now supported in this Library.

•  Library name changed from "STM8 TouchSensing Library" to "STM8S TouchSensing Library".

•  Update with latest "STM8S/A Standard Peripherals Driver" version.

 

Content

•  STM8S TouchSensing Driver V2.5.0. (Release Notes)

•  STM8S TouchSensing Examples V2.5.0. (Release Notes)

•  STM8S/A Standard Peripherals Driver V2.1.0. (Release Notes)

 

Development Toolchains and Compilers

•  ST Visual Develop (STVD) v4.3.4.

•  Cosmic STM8 C Compiler v4.3.9.

•  Raisonance Ride7/RKit-STM8 v2.51.13.

•  IAR Embedded Workbench for STM8 v1.41.

 

Supported Devices and EVAL boards

•  STM8/128-EV/TS board (composed of MB631 board and STM8SxxxK-TS1 daughter board).

 

Known Limitations

•  None.
